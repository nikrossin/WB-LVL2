Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]
срез массива возвращает слайс - [:] это способ сделать слайс на основе массива
b это слайс внутри которого лежит указатель на массив a, слайс "видит" элементы с 1 по 3 включительно
cap = 4, поскольку до конца массива а есть еще один свободный элемент на расширение слайса, 
без выделения памяти под новый массив

```
