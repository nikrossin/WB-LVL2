Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2 \n 1

Оператор defer помещает вызов функции в стек(LIFO - последний зашел - первым вышел).
defer выполняется после return, то есть исполняется реагируя на него.
Return копирует значения в память (область возврата), поскольку выходит из области работы функции.
AnotherTest функция копирует значение переменнй х==1 в область памяти возврата и после выполняется отложенная функция,
которая увеличивает значение локальной переменной внутри функции Х на 1, однако в памяти уже записана 1.
test() имеет именованный возврат (x int). Return выполняется как "просто вернуть то что в именнованной области памяти",
после выполняется отложенная функция, которая обращается к именнованной области возврата и увеличивает значение на 1,
в результате получаем 2.

```
